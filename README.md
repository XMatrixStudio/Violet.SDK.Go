# Violet SDK for Golang



## Installation

### Current Version - v3.0.5

**go.mod**

```
require gopkg.in/xmatrixstudio/violet.sdk.go.v3 v3.0.5
```

**go get**

```
go get -u gopkg.in/xmatrixstudio/violet.sdk.go.v3
```

### Old Version - v2.0

**go get**

```
go get -u github.com/XMatrixStudio/Violet.SDK.Go
```
